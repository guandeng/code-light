## v1.2.0 更新内容

### 🆕 S3 兼容云同步
- 新增 S3 对象存储作为第二套配置同步后端，与 WebDAV 互斥切换
- 自实现 AWS Signature V4 签名，无需第三方 SDK
- 内置 7 个服务商预设：AWS S3、Cloudflare R2、MinIO、阿里云 OSS、腾讯云 COS、华为 OBS、自定义
- 支持连接测试、手动上传/下载、自动同步

### 🆕 Sparkle 自动更新
- 集成 Sparkle 框架，支持自动检查更新和一键安装新版本
- 不再依赖手动 gh CLI 检查

### 🔒 SQLite 持久化
- 工作统计数据迁移到 SQLite 本地存储，提升查询性能
- 新增危险命令黑名单机制

### 🐛 修复
- 修复 Hook 变量在部分场景下未正确传递的问题
- 调整设置窗口中统计文档的高度以适应主视图
